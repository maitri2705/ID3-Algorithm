The ID3 decision tree algorithm is implemented using Java in Netbeans IDE.

The program is complied perfectly in Netbeans IDE. 
We can also used command prompt for compliation by using following commannd:

 >javac *.java

The above command will compile all java file.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
   ****** The main() method is written in MainClass.java file ***********
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

We need to give 4 arguments in command line.
	(1) complete path of training_set.csv
	(2) complete path of validation_set.csv
	(3) complete path of testing_set.csv
	(4) value of pruning factor

The Command should look like this:

 >java MainClass pathOfTrainingDataSet pathOfValidationDataSet pathOfTestingDataSet valueOfPruningFactor
